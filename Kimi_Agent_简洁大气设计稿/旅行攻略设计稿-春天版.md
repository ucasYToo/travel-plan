# 首尔旅行攻略 - 春天版设计稿

## 设计方向

### 风格定位
- **季节感**：春天、清新、温暖
- **设计语言**：苹果风格（Apple Design）
- **情绪**：轻松、愉悦、可爱

### 核心特点
1. **低饱和度配色** — 柔和不刺眼
2. **大圆角设计** — 参考iOS风格
3. **玻璃拟态效果** — 半透明、毛玻璃质感
4. **可爱元素** — 小图标、插画点缀
5. **充足留白** — 呼吸感强

## 组件设计

### 圆角规范
- 大卡片：20px
- 小卡片：16px
- 按钮：14px
- 标签：20px（胶囊形）
- 输入框：12px

### 阴影规范
- 卡片阴影：`0 4px 20px rgba(0,0,0,0.06)`
- 悬停阴影：`0 8px 30px rgba(0,0,0,0.1)`

### 可爱元素
- 地点标记：小花朵/小叶子图标
- 酒店图标：小房子带爱心
- 交通图标：小飞机/小火车带笑脸
- 装饰：角落小花、飘落花瓣

## 设计亮点

1. **春天色彩** — 薄荷绿、樱花粉、奶油黄，清新温暖
2. **苹果风格** — 大圆角、毛玻璃、精致阴影
3. **可爱细节** — 小图标带表情，角落装饰
4. **低饱和度** — 柔和舒适，长时间使用不疲劳
5. **充足留白** — 界面呼吸感强
